All code in here is 100% readable, no jQuery, react, bootstrap... just pure JavaScript, html, and CSS.

This extension has the following features.
	let’s you save URLs from videos and give them descriptions (names, categories, comment).
	Scrambles user data to save to storage
	opens tabs in Incognito when going through popup.
	closes incognito when browser is in idle
	blocks side content when opening tabs containing the video.
	let’s you save skips for parts of video you don’t want to see. (hold 'c' and move the seek bar over parts you want to skip)
	lets you download/uplaod back your data.
